You can use Mic-1 MMV's IJVM Assembler to practice writing assembly language programs, which you can then test on the simulator. Read the JAS Language Specification for an explanation of the structure of an IJVM program.

  • Using your favorite text editor, create a text file which contains your Java Assembly Language (JAS) program. We have adopted the convention of using .jas as the file name suffix to indicate that it is a "Java Assembly" language file.

    You may want to create a copy of echo.jas or add.jas and modify it to suit your needs.

  • Launch the simulator
  • Select File | Assemble / Load JAS file .... Navigate to your file and select it.
  • The JAS Assembler View window should appear. If assembly complete appears in the text area you can click the Load button to read the IJVM program into the Method Area; otherwise, if error messages appear in the assembly window, correct the errors and try again.
Your IJVM program is now loaded into the Mic-1 MMV simulator. You can view the program in the Method Area. IJVM execution can be observed best by running the simulator at IJVM or Prog speed.

If you modify your JAS program, you can reassemble and reload the current IJVM program by using Assemble/Load | Current JAS Assemble/Load.
